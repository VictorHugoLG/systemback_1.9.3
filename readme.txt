Este Pacote de Instalação contém os pacotes de instalação do Systemback Debian.
Esses pacotes são compatíveis com as seguintes distribuições Linux:
 - Ubuntu 17.10 (Artful Aardvark)
 - Debian 9.0 (Stretch)

Para instalar o programa basta executar o script do instalador 'install.sh' com privilégios de root no terminal, assim:

 sudo ./install.sh

ou

 sudo sh install.sh

Se você deseja instalar o programa com pacotes de depuração, basta executar o script do instalador 'install.sh' com a opção '-d':

 sudo ./install.sh -d

ou

 sudo sh install.sh -d

Se a instalação for bem-sucedida, a seguinte mensagem será exibida:

 A instalação do Systemback foi bem-sucedida.

Se a instalação falhar, você receberá a seguinte mensagem de erro:

 Falha na instalação do Systemback!
